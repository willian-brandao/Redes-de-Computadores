# Modelo de Referência de Rede

## Aplicações e Dados

Aplicações são usadas para atender vários requisitos de pessoas, como acesso de páginas web, jogos online, e reprodução de vídeo online. 

Informações são geradas junto com as aplicações. Textos, fotos, e vídeos são todos os modos de apresentação de informação. 

## Implementação de Aplicação - Dados 

Geração de Dados - Na área de computação, dado é o operador de todos os tipos de informação. 

Transmissão de Dados - Dados gerados por grande parte das aplicações precisam ser transmitidos entre dispositivos.

## Modelo de Referência de Redes e Protocolos Padrão 

|   Camada de Rede      | Descrição                                                                                                                  |
|-----------------------|----------------------------------------------------------------------------------------------------------------------------|
| Camada de Aplicação   | Provê interface para aplicações                                                                                            |
| Camada de Apresentação| Estabelece, gerencia, e termina sessiões entre comunicações das partes                                                     |
| Camada de Transporte  | Estabelece, mantém, e cancela processo de dados fim a fim, controla a velocidade de transmissão e ajusta sequência de dados|
| Camada de Rede        | Define endereços lógicos e transfere dados da origem para o destino                                                        |
| Camada de Enlace      | Encapsula pacotes em frames, transmite os frames em modo P2P ou P2MP e implementa checagem de erros                        |
| Camada Física         | Transmite bitstreams sobre mídias de transmissão e define especificações elétricas e físicas                               |
