# Componentes de Redes

## Funções de hosts 

Todos os dispositivos conectados em uma rede e que participam dela, são classificados como hosts. Os hosts podem ser conhecidos como  clientes. O termo se refere a dispositivos na rede que foram atribuidos um número a fim de comunicação. Esse número é o que o identifica dentro de uma rede. O número é conhecido como endereço de Protocolo Internet (IP). Um endereço IP identifica o host e a rede na qual o host está ligado. 

Servidores são computadores com software que permite que eles provêm informação, como e-mail ou páginas web, para outros dispositivos na rede. Cada servidor requer um software de server separado. Por exemplo, um servidor requer um software de servidor web para fornecer serviços web para a rede. Um computador com software de servidor pode prover serviços simultâneos para vários clientes diferentes.

Um exemplo de software cliente é o navegador web, como Chrome e Firefox. Um único computador pode rodar diversas tarefas de software cliente. Exemplo: uso simultâneo de browser internet, envio de e-mails e ouvir música. 

|                  Tipo               | Descrição                                                                                                                                                           |
|-------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Email                               | O servidor de e-mail roda o software de servidor de email. Clientes usam os software de cliente de e-mail, como Microsoft Outlook para acessar o servidor de e-mail.|
| Web                                 | O servidor web roda o software de servidor web. Os clientes usam o software de navegador, como Microsoft Edge para acessar páginas web                              |
| Arquivos                            | O servidor de arquivo armazena arquivos corporativos e de usuários em uma localização centralizada. Os dispositivos clientes acessam esses arquivos com software de cliente como o Windows File Explorer |

## Rede Ponto a Ponto (Peer-to-Peer)

Softwares de cliente e servidor normalmente rodam em máquinas separadas, mas é possível que um computador seja usado para as duas funções ao mesmo tempo. Em pequenos negócios e residências, computadores funcionam como servidores e clientes na rede. Esse tipo de rede é chamado de rede ponto a ponto ou peer-to-peer.

Vantagens:
* Fácil de configurar
* Menos complexa
* Baixo Custo porque os dispositivos de rede e servidores dedicados não são necessários.
* Pode ser usada para tarefas simples como transferência de arquivos e impressoras compartilhadas. 
Desvantagens:
* Não tem administração centralizada.
* Não há segurança.
* Não é escalável.
* Todos os dispositivos podem agir como clientes ou servidores que pode diminuir a performance deles.

## Dispositivos Finais

Os dispositivos de rede que são mais familiares as pessoas são os dispositivos finais. Para distinguir um dispositivo final de outro, cada dispositivo final na rede tem um endereço. Quando um dispositivo inicia a comunicação, ele usa o endereço de destino para especificar para onde enviar a mensage. Um dispositivo final é a origem ou destino de uma mensagem transmitida através da rede.
